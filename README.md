A minimal framework around the WebGL API.

![Screenshot](/example/screenshot.png?raw=true)

## Default vertex attributes
- aVertexPosition
- aVertexIndex
- aVertexNormal
- aVertexColor
- aTextureCoord

## Default matrix uniforms
- uProjectionMatrix
- uModelMatrix
- uModelViewMatrix
- uNormalMatrix

## Installation

Make sure you have [Node.js](http://nodejs.org/) installed.

$ npm install

## Usage

$ npm start

$ npm run lint

$ npm run dist

$ npm run deploy
